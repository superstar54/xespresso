Book-DFT-Quantum Espresso¶

Introduction¶

The original dft-book was written for VASP by John Kitchin (https://github.com/jkitchin/dft-book). In this book, we are rewriting some of that book for use with Quantum Espresso, ASE and xespresso module (https://github.com/superstar54/xespresso).

Atomic Simulation Environment¶

The Atomic Simulation Environment (ASE) is a set of tools and Python modules for setting up, manipulating, running, visualizing and analyzing atomistic simulations. For the introduction of ASE , please visit https://wiki.fysik.dtu.dk/ase/index.html

import matplotlib.pyplot as plt

Density of states (DOS)¶

The density of states (DOS) gives you the number of electronic states (i.e., the orbitals) that have a particular energy. For insulator, the number of electronic states (bands) to be calculated is the number of valence bands (nbnd). For CO, nbnd = (4+6)/2 = 5. More bands can be setted, if we want to see the dos above the fermi energy.

In [27]:
from xespresso.dos import DOS
import matplotlib.pyplot as plt

pseudopotentials = {'O': 'O.pbe-n-rrkjus_psl.1.0.0.UPF',
                    'C': 'C.pbe-n-rrkjus_psl.1.0.0.UPF'}
calc = Espresso(label = 'molecule/co-dos/co-dos',
                pseudopotentials = pseudopotentials,
                ecutwfc = 30,
                nbnd = 7,
                kpts = (1, 1, 1),
                queue = queue)
co.calc = calc
e = co.get_potential_energy()
print('Energy  = {0} eV'.format(e))
calc.read_results()
fermi = calc.get_fermi_level()
calc.nscf(queue = queue, type = 'dos', kpts = (1, 1, 1))
calc.nscf_calculate()
calc.post(queue = queue, package = 'dos', Emin = fermi - 20, Emax = fermi + 10, DeltaE = 0.1)
dos = DOS(calc)
dos.read_dos()
dos.plot_dos(Emin = -20, Emax = 10)
plt.xlabel('Energy - $E_f$ (eV)')
plt.ylabel('DOS')
plt.savefig('images/co-dos.png')
